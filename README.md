Pastikan telah mengistall docker, docker compose, dan git 
Lakukan download dari repository ini atau 
`git clone https://github.com/irawand07/tcc-docker-compose`
Jalankan perintah 
`docker-compose up -d`
Jika sudah selesai buka browser ketikkan  di url
`http://localhost/`
